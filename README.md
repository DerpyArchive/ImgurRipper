# ImgurRipper
Node.js Imgur album ripper

## How to use

### Manual
1. ``git clone https://github.com/ohlookitsderpy/ImgurRipper``
2. ``cd ImgurRipper``
3. ``node main.js``

### Automatic
1. Download the exe from the releases page
2. Run the exe
